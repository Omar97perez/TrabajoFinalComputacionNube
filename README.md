## Build Setup
- Cliente.
  - Instalar dependencias: npm run dependencies:client.
  - Correr cliente: npm run client.
  - Correr webpack: npm run webpack.
- Servidor.
  - Instalar dependencias: npm run dependencies:server.
  - Correr servidor: npm run server.
